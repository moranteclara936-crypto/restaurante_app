# Sistema de Gestión de Restaurante

## Descripción
Este proyecto consiste en una aplicación para la gestión de un restaurante, desarrollada en Python utilizando Programación Orientada a Objetos (POO). El sistema está organizado en módulos para facilitar su mantenimiento y comprensión.

## Objetivo
Desarrollar un sistema que aplique los principios de la Programación Orientada a Objetos y permita organizar correctamente el código en diferentes módulos.

## Estructura del proyecto

- modelos/: Contiene las clases del sistema.
- servicios/: Contiene la lógica y funciones del sistema.
- main.py: Archivo principal para ejecutar la aplicación.

## Requisitos

- Python 3 o superior.

## Ejecución

1. Abrir el proyecto en Visual Studio Code.
2. Ejecutar el archivo `main.py`.
